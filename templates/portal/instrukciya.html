IT: Інструкція з додавання статей в адмін панелі

Опубліковано: 12 січня 2022 р.

Добавление изображений

- В панеле редактора выбрать "Зображення"

- Перейти на вкладку "Надiслати" -> "Обзор" -> Выбрать нужное изображение для загрузки -> Нажать кнопку "Надiслати на сервер" -> При необходимости изменить параментры -> Нажать "Ок".

- В случае, если изображение должно открываться по клику в новой вкладке, перейти на вкладку "Посилання" и в списке "Цiль" выбрать "Нове вiкно".

В редакторе изображение будет прижато к одному из краев, но на сайте изображения будут размещаться по центру страницы.

Загрузка файлов

- В панеле редактора выбрать "Вставити/Редагувати посилання"

- Перейти на вкладку "Надiслати" -> Выбрать нужный файл -> Нажать кнопку "Надiслати на сервер".

- По умолчанию загруженный файл будет открываться на чтение в браузере. Если файл необходимо будет скачивать локально, нужно перейти на вкладку Додаткове и установить галочку в поле Завантажити як файл.

- В поле "Вiдображуванний текст" ввести название ссылки, которое будет отображаться на сайте.

- Поля "Протокол" и "URl" не изменять

Добавление ссылок

- В панеле редактора выбрать "Вставити/Редагувати посилання"

- В поле "Вiдображуванний текст" ввести название ссылки, которое будет отображаться на сайте.

- В поле "URL" вставить ссылку. Поле "Протокол" измениться автоматически.
